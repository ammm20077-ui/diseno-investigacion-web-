🖼️ Clasificación de diseños cuantitativos

1.1 🧪 Diseños Experimentales (causa y efecto)

Permiten establecer relaciones de causa y efecto controlando la intervención sobre una variable independiente.

  • 👥 Grupo experimental y grupo control
  • 🎲 Asignación aleatoria
  • 🛡️ Control de variables externas
  • 📏 Medición de variable dependiente

🎥 Diseños experimentales vs no experimentales

⏳ 1.1.1 Diseños Pre-experimentales

No emplean asignación aleatoria ni control riguroso. Se usan como aproximación inicial con validez interna limitada.

  • ❌ Falta grupo de control equivalente
  • 🧪 Manipulación de variable independiente
  • 📏 Medición de variable dependiente
  • ⚠️ Factores externos pueden influir
Ejemplo: evaluar rendimiento después de una intervención sin medición previa ni grupo comparación.
⚖️ 1.1.2 Diseños Cuasi-experimentales

Buscan causalidad, pero no usan asignación aleatoria. Los grupos existen antes del estudio.

  • ✅ Útiles si aleatorizar no es práctico o ético
  • 🌍 Estudian el mundo real
  • 📌 Evidencia fuerte si el diseño es sólido
📌 Tipos (cuasi-experimentales)
  • Grupo de control no equivalente
  • Series temporales interrumpidas
  • Diseños de cohortes
  • Diseños de casos únicos
✅ 1.1.3 Diseño Experimental Verdadero

Manipula variables y controla otras variables mediante asignación aleatoria.

  • 🧪 Manipulación intencional
  • 🎲 Aleatorización
  • 🛡️ Control de variables extrañas

1.2 🗂️ Diseños No Experimentales

No manipulan variables ni asignan aleatoriamente. Observan el fenómeno en su contexto natural.

📷 Transeccionales / Transversales

Datos en un solo momento para describir e interrelacionar variables.

⏳ Longitudinales

Recolectan datos a través del tiempo para analizar cambios y tendencias.

🗂️ Retrospectivos

Usan datos históricos para estudiar eventos pasados.

📈 Prospectivos

Siguen a los participantes hasta que ocurren los eventos (hacia adelante).

📌 Clasificación transeccional

Exploratorio, descriptivo y correlacional.

🗣️ Diseños Cualitativos

Comprenden fenómenos humanos desde la perspectiva de los participantes.

✨ Enfoque cualitativo

  • ❌ No hay intervención
  • ❌ No prueba hipótesis causales
  • 🌍 El contexto importa
  • 🎯 Muestra intencional
  • 📝 Datos narrativos

🖼️ El ciclo de la investigación cualitativa

2.1 🧠 Teoría Fundamentada

Genera teoría desde datos recopilados y analizados de forma sistemática.

  • 📥 Recolección de datos
  • 🔤 Codificación y categorías
  • 🔄 Análisis continuo
  • 🎯 Teoría emergente

2.2 🌐 Etnográficos

Estudian culturas y comunidades con observación participante e inmersión prolongada.

  • 👀 El investigador pasa tiempo con el grupo
  • 🧩 Observa prácticas y rituales
  • 🤝 Participa en actividades
  • 🎯 Interpreta valores y significados
Tip: escribe notas de campo y analiza patrones de interacción.
2.3 📖 Narrativos

Se centran en historias personales y experiencias: relatos, diarios, entrevistas, etc.

  • 📚 Relatos de vida
  • 🧠 Analiza significados
  • 🧭 Considera estructura y temas
2.4 🔁 Investigación-Acción

Enfoque participativo para resolver problemas prácticos con ciclos: planificación → acción → observación → reflexión.

  • 🤝 Participantes e investigadores trabajan juntos
  • 🛠️ Se diseñan soluciones
  • 📊 Se evalúan resultados
  • 🔁 Ciclos para mejorar continuamente

🔀 Diseños Mixtos

Combinan cuantitativo y cualitativo en un mismo estudio.

Idea clave

Se combina el rigor numérico con la riqueza interpretativa.

  • ✅ Usa ambos enfoques
  • ⏱️ Secuencial o simultáneo
  • 🧩 Requiere claridad de objetivos
  • 💰 Más tiempo y recursos

🖼️ Integración de enfoques mixtos

3.1 🧾 Secuencial explicativo (Cuantitativo → Cualitativo)
  1. 📊 Primero: medir y analizar cuantitativo
  2. 🗣️ Luego: cualitativos para explicar o ampliar resultados
Ejemplo: medir estrés académico (encuesta) y luego entrevistar para conocer causas.
3.2 🔍 Secuencial exploratorio (Cualitativo → Cuantitativo)
  1. 🗣️ Comienza con cualitativos para explorar el fenómeno
  2. 📊 Luego aplica cuantitativos para generalizar hallazgos
Ejemplo: explorar habilidades digitales con entrevistas; después encuesta a mayor escala.
3.3 ⚡ Concurrente

Se trabajan componentes cuantitativos y cualitativos al mismo tiempo.

Ejemplo: evaluar calidad de vida de mascotas: observación cualitativa y análisis cuantitativo.
3.4 📦 Anidados

Un componente está "encapsulado" dentro del otro (jerarquía interna).

Ejemplo: productividad (general) encapsula el efecto del sueldo (subproceso).
